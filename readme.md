
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/fastapi-user-registration.git
   cd fastapi-user-registration

2. Make sure you have the following installed:

        - Python (>= 3.7)
        - PostgreSQL
        - MongoDB
